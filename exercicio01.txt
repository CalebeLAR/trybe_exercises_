1.Crie uma pasta em seu computador;
comando: 
~ mkdir gitExercicioS1D2

2.Acesse a pasta que você acabou de criar;
comando: 
~ cd gitExercicioS1D2

3.Inicie o versionamento por meio do comando git init;
comando: 
~ git init

4.Abra a pasta no VS Code por meio do comando code .;
comando: 
~ code .

5.Crie um arquivo chamado README.md;
comando: 
~ echo #exercício seção01 dia02 > README.md

6.Adicione o arquivo novo em staging e realize um commit, como em git add . e git commit -m "Descreva a alteração realizada";
comando: 
~ git add .
~ git commit -m 'exercicio seção 01 dia 02/ README.md'

7.Crie uma nova branch por meio do comando git checkout -b, como em git checkout -b adiciona-readme;
comando: 
~ git checkout -b adiciona-readme

8.Faça uma alteração em README.md e, depois, realize um novo commit, como em git add . e git commit -m "Alterando README";
comando: 
~ git add README.md
~ git commit -m 'Alterando README"

9.Retorne para a branch main e, em seguida, realize o merge das alterações por meio do comando git merge nome-da-branch, como em git merge adiciona-readme. Retorne para a branch main e realize o merge das alterações.
comando: 
~ git checkout main
~ git merge adiciona-readme // README.md | 4 +++-
