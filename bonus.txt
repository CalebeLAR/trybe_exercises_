1.Se você utiliza o Linux sudo apt-get install cmatrix. Para pessoas usuárias de macOS brew install cmatrix no terminal. Depois, execute o comando cmatrix. Aperte ctrl+c para voltar ao terminal.
comando:
~ sudo apt-get install cmatrix
~ cmatrix

2.No sistema Linux, execute o comando sudo apt-get install fortune. No macOS brew install fortune. Após a instalação, crie um arquivo de texto chamado fortune.txt que contenha a sua sorte do dia. Utilize apenas uma linha de comando.
comando:
~ sudo apt-get install fortune
~ fortune.txt > fortune.txt // You will be successful in love.

3.Conte o número de palavras que a frase da sua sorte do dia tem.
comando:
~ wc -w fortune.txt // 6

4.Execute o comando sudo apt-get install sl em um terminal Linux, ou brew install sl em um terminal macOS. Após a instalação, execute o comando sl. Agora, tente sl -f.
comando:
~ sl // treim
~ sl -f // treim

5.No sistema Linux, execute o comando sudo apt-get install cowsay, ou brew install cowsay no macOS. Após a instalação, execute o comando cowsay e algo que você queira falar. Agora, faça a vaquinha dizer a frase que está gravada no arquivo fortune.txt.
comando:
~ cowsay trybe // vaquinha com um balão de comentario ectrito trybe
~ tail fortune.txt | cowsay

6.Descubra os fatores primos usando o comando factor e, em seguida, o número 42.
comando:
~ factor 42 // 42: 2 3 7

7.Veja como fica a sua sorte do dia ao contrário. Para isso, utilize o comando rev.
comando:
~ tail fortune.txt | rev // .evol ni lufsseccus eb lliw uoY

8.Execute o comando telnet towel.blinkenlights.nl e espere alguns segundos.
comando:
~ telnet towel.blinkenlights.nl // starwars

9.Passe o desenho da vaquinha falando trybe para um arquivo vaquinha.txt
comando:
~ cowsay trybe | cat vaquinha.txt
