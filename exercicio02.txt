1.	Crie a pasta unix_tests_skills e navegue até ela.
comando:
~ mkdir unix_tests_skills
~ cd unix_tests_skills
~ pwd

2.	Crie um arquivo de texto pelo terminal com o nome skills2.txt e adicione os valores Internet, Unix e Bash, cada um em sua linha.
comando:
~ touch skills2.txt
~ echo "Internet" >> skills2.txt
~ echo "Unix" >> skills2.txt
~ echo "Bash" >> skills2.txt

3.	Adicione mais 5 itens à sua lista de skills e, depois, imprima a lista ordenada no terminal.
comando:
~ cat >> skills2.txt // ENTER javaScript python typeScript react node.js redux
~ sort < skills2.txt

4.	Conte o número de linhas do arquivo skills2.txt.
comando:
~ cat skills2.txt | wc -l

5.	Crie um arquivo chamado top_skills.txt usando o skills2.txt. Ele deve conter as 3 primeiras skills em ordem alfabética.
comando:
~ sort < skills2.txt | head -n 3 > top_skills.txt

6.	Crie um novo arquivo chamado phrases2.txt pelo terminal e adicione algumas frases de sua escolha.
comando:
~ touch phrases2.txt
~ echo "The quick brown fox jumps over the lazy dog." > phrases2.txt
~ echo "Quick fox jumps nightly above wizard." >> phrases2.txt
~ echo "The quick onyx goblin jumps over the lazy dwarf." >> phrases2.txt

7.	Conte o número de linhas que contêm as letras br.
comando:
~ grep br phrases2.txt | wc -l // 1

8.	Conte o número de linhas que não contêm as letras br.
comando:
~ grep -v br phrases2.txt | wc -l // 2

9.	Adicione dois nomes de países ao final do arquivo phrases2.txt.
comando:
~ echo "Japão" >> phrases2.txt
~ echo "Austrália" >> phrases2.txt

10.	Crie um novo arquivo chamado bunch_of_things.txt com os conteúdos dos arquivos phrases2.txt e countries.txt.
comando:
~ cp ../unix_tests_search countries.txt bunch_of_things.txt
~ cat phrases2.txt >> bunch_of_things.txt

11.	Ordene o arquivo bunch_of_things.txt.
comando:
~ sort bunch_of_things.txt -o bunch_of_things.txt
